# 🚀 30-Day Vibe Code Sprint

> One polished, useful app — shipped every single day. Fully automated with GitHub Actions + Claude AI + GitHub Pages.

---

## 🗺️ How it works

```
Every day at 6:00 AM IST (auto) or on-demand (manual trigger):

1. GitHub Actions wakes up
2. Determines today's day number from SPRINT_START_DATE
3. Calls Claude API → generates complete React + Vite app
4. Builds it with Vite
5. Deploys to GitHub Pages (free hosting)
6. Logs the live URL to deployments.md
7. Regenerates the portal homepage
```

**Stack:** React 18 · Vite · Tailwind CSS · GitHub Actions · GitHub Pages
**Cost:** $0 hosting · Only Claude API usage (~$0.02–0.10 per app)

## ⚡ One-Time Setup (10 minutes)

### Step 1 — Fork / clone this repo
```bash
git clone https://github.com/YOUR_USERNAME/vibe-sprint.git
cd vibe-sprint
```

### Step 2 — Enable GitHub Pages
1. Go to your repo → **Settings** → **Pages**
2. Source: **GitHub Actions**
3. Save

### Step 3 — Add repository secrets
Go to **Settings → Secrets and variables → Actions → New repository secret**

| Secret name | Value |
|---|---|
| `ANTHROPIC_API_KEY` | Your key from console.anthropic.com |
| `SPRINT_START_DATE` | Today's date in `YYYY-MM-DD` format (e.g. `2026-03-15`) |

### Step 4 — Enable GitHub Actions
Go to **Actions** tab → click **"I understand my workflows, go ahead and enable them"**

### Step 5 — Run Day 1 manually
1. Go to **Actions** → **🚀 Daily Vibe App Deploy**
2. Click **Run workflow**
3. Leave "day" blank (auto-detects Day 1) → **Run workflow**

That's it! From Day 2 onwards it runs automatically at 6 AM IST every day.

## 🎮 Manual controls

### Deploy a specific day
```
Actions → Daily Vibe App Deploy → Run workflow → enter day number (1–30)
```

### Regenerate an app
Check `force_regenerate = true` when triggering manually. Claude will rewrite the app from scratch.

### Run locally (for testing)
```bash
cd scripts
npm install
ANTHROPIC_API_KEY=sk-ant-xxx DAY=1 node generate-app.js
DAY=1 node build-app.js
DAY=1 GITHUB_REPOSITORY=username/vibe-sprint node update-log.js
```

## 💡 Tips

- **AI apps (days 15, 19, 25, 28):** Users of those apps need to enter their own Anthropic API key in the app's settings. The key is never sent anywhere except the Anthropic API.
- **Costs:** Each Claude API call uses ~3,000–6,000 tokens. At Sonnet pricing, that's roughly ₹1–5 per app generated.
- **Build failures:** If a day's build fails, re-trigger manually with `force_regenerate=true`. Claude will write a fresh version.
- **Customise the schedule:** Edit the `cron` line in `daily-deploy.yml` to change the deployment time.
